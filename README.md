# Campus Food Pantry System

A C++ Object-Oriented Programming project designed to manage a university food pantry.

## Features

- Inventory Management
- Student Food Requests
- Request History Tracking
- Random Food Donations
- Inventory Sorting
- File Save/Load Functionality
- Unit Testing
- Linked List Implementation
- Inheritance and Polymorphism
- Big Three Implementation

## Technologies

- C++
- STL Vector
- File I/O
- Linked Lists
- Object-Oriented Programming

## Classes

- Item
- User
- Student
- Staff
- Pantry
- RequestHistory

## Build

```bash
make
./main
```

## Learning Outcomes

This project demonstrates:

- Data Structures
- Dynamic Memory Management
- OOP Design
- File Persistence
- Software Testing
